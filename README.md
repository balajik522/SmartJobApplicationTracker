# Smart Job Application Tracker

Smart Job Application Tracker is a full-stack web application that helps job seekers organize and manage their job search efficiently.

## Features

- Track job applications
- Monitor application status
- Store company and interview details
- Set reminders for follow-ups
- View application statistics through a dashboard
- Resume upload and management
- Analytics and notifications

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Spring Boot
- Spring Data JPA
- MySQL
